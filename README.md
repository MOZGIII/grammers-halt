# grammars-hang

Pass the settings via env vars or
just edit [`crates/bin/main/src/main.rs`](crates/bin/main/src/main.rs)
and hard-code them locally.

Sample `.env`:

```bash
export TG_API_ID=""
export TG_API_HASH=""
export TG_BOT_TOKEN=""

export TG_CHANNEL_ID=""
export TG_CHANNEL_ACCESS_HASH=""
```
